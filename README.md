# 🧠 HR Analytics Dashboard | Workforce Insights

## 📌 Overview

This project delivers an **interactive HR Analytics Dashboard** designed to uncover actionable insights about workforce distribution, employee performance, and compensation trends across multiple locations.

The dashboard transforms raw HR data into **clear, decision-ready insights** that support strategic workforce planning and organizational growth.

---

## 🎯 Business Problem

Organizations often struggle to:

* Understand salary distribution across departments
* Identify performance gaps between locations
* Analyze workforce demographics effectively
* Make data-driven HR decisions

This project addresses these challenges by providing a **centralized analytical view of HR data**.

---

## 💡 Solution

An interactive dashboard that enables users to:

* Compare performance across locations and departments
* Analyze salary structure (avg, min, max, total)
* Understand workforce demographics (age, experience)
* Drill down from overall view → department → job role

## 🧱 Data Model

The dataset includes:

* Location → Department → Job hierarchy
* Employee count & aggregation levels
* Salary metrics (Avg, Total, Min, Max)
* Demographics (Age, Years of Experience)
* Performance indicators (2019–2020)
